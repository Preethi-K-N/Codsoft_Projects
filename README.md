### 1Ô∏è‚É£ Titanic Survival Prediction üö¢
* **Objective:** Predict the survival of passengers on the Titanic based on demographics and ticket class.
* **Key Techniques:** Data Cleaning, Feature Engineering, Binary Classification.
* **Outcome:** Built a model that accurately predicts survival probabilities.

### 2Ô∏è‚É£ Movie Rating Prediction üé¨
* **Objective:** Analyze historical movie data to predict user ratings based on genre, director, and cast.
* **Key Techniques:** Regression Analysis, Data Preprocessing.
* **Outcome:** A regression model providing insights into factors influencing movie success.

### 3Ô∏è‚É£ Iris Flower Classification üå∏
* **Objective:** Classify Iris flowers into three distinct species (Setosa, Versicolor, Virginica) using sepal and petal measurements.

* **Outcome:** High-accuracy classification suitable for botanical analysis.

### 4Ô∏è‚É£ Sales Prediction üìà
* **Objective:** Forecast product sales by analyzing advertising budgets across TV, Radio, and Newspaper.

### 5Ô∏è‚É£ Credit Card Fraud Detection üí≥
* **Objective:** Detect fraudulent transactions in a highly imbalanced dataset.

* **Outcome:** A robust system capable of distinguishing between genuine and fraudulent activities.
